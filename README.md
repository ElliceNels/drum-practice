# drum-practice
This project involves the development of a practice-tracking application for drummers, designed to help users monitor and improve their tempo accuracy. Tempo control is a fundamental skill for musicians, particularly drummers, who are responsible for maintaining the pacing and feel of live performances. 

The proposed application will analyze practice sessions and provide feedback on the discrepancy between a drummer’s intended tempo and their actual performance. By tracking these metrics over time, the application will establish baseline statistics and highlight days of higher proficiency or weaker performance
